Prism AI

Intelligent multi-model AI chat with SAT quiz, file analysis, offline-first sync, and a powerful admin panel. One app, every AI tool you need.

Launch App Read the Docs
🤖

Multi-Model Chat

Chat with any AI model via OpenRouter or your local server. Switch models on the fly or run multiple models in parallel with Multi-Grade mode.

⚖️

Multi-Grade Mode

Run multiple AI models simultaneously and have them grade each other's responses. Configurable weights for helpfulness, accuracy, clarity, and speed.

📝

SAT Quiz

Upload SAT question banks as PDFs or paste text. Adaptive quiz mode with momentum tracking, anti-click-through, two-pass review, and detailed analytics.

📁

File Analysis

Upload documents and images. AI summarizes content, extracts key points, and answers questions about your files. Supports PDFs, text, and images.

🎤

Voice Input

Speak your prompts with built-in speech-to-text. Visual waveform feedback and one-tap recording right from the chat input.

☁️

Cloud Sync

Everything syncs to Supabase when online. Works 100% offline with localStorage/IndexedDB. Your data is there when you come back.

🔧

Admin Panel

Admin-controlled shared database. Manage users, configure Supabase credentials, monitor storage, set global defaults, and view audit logs.

📱

Installable PWA

Install Prism AI on your device. Works like a native app with offline support, push-ready architecture, and a custom splash screen.

Getting Started

  1. Open chat.html and add your API key in Settings.
  2. Select your preferred single model.
  3. Send a message to start your first chat.

Chat Modes

Single runs one model response. Multi-Grade runs multiple models and grades outputs.

Use Multi-Grade when you need stronger reliability or side-by-side quality checks.

Attachments

Attach files using the upload button in the composer. File data is sent to the model as hidden context (not dumped into your visible chat bubble).

  • Text files are extracted and included as text context.
  • Images (including .ico) are passed as data URLs.
  • Pending upload chips let you remove files before sending.

Voice Input

Use the mic button to start/stop speech-to-text. Live transcript appears above the composer before sending.

Offline + Sync

Prism works offline with local storage and can sync with Supabase when online.

  • Offline mode uses your configured local/offline model.
  • When reconnected, data can sync back to cloud storage.

Settings Highlights

  • Model selection (single, multi, grader)
  • Temperature and max tokens
  • Transparency mode and debug console
  • Theme toggle, export/import, storage meter

Troubleshooting

  • If model calls fail: verify API key and selected model availability.
  • If voice input fails: ensure browser mic permission is allowed.
  • If sync fails: confirm Supabase credentials in supabase.config.local.js.
